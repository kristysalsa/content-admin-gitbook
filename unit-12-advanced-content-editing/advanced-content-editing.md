# Advanced content editing

Page layout in GovCMS is limited to predefined and preconfigured regions. The layout cannot be easily changed without some theme development. However, thanks to the Panelizer module in GovCMS, more complex layouts can be managed by content editors. The Panelizer module allows a content administrator to create custom layouts for multiple uses.
